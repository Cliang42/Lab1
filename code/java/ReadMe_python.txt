======================================
explanation of the python code files
======================================

Build_Index.py: build r-tree index given data points

CusPoint.py: object used to store data points

Node.py: the node object that used as nodes in r-tree

ReadPoints.py: read points from files, save results to files

RTree.py: implementation of r-tree

Skyline.py: B&B, BNL implementatin

Skyline_Test.py: test of ranked skyline query
